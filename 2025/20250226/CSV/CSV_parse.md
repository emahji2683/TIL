# CSV.parseとは
文字列として受け取ったcsvを配列に変換することができる。

> このメソッドは文字列を簡単にパースすることができます。ブロックを与えた場合は、ブロックにそれぞれの行を渡します。ブロックを省略した場合は、配列の配列を返します。  

# 使用方法
以下の通り、parseメソッドの引数にパースしたい文字列を与える。

```
CSV.parse(文字列)
```
