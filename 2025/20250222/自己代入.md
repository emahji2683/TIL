# 自己代入(||=)とは
> 左辺の変数が、nilもしくはfalseのとき||=の右辺にある値を代入するという意味です。

# 具体例
変数に何らか値が存在する時、当初の値がそのまま出力される。
変数に値が存在しない時、右辺の数字が出力される。

```
# 存在する場合
b = 10
b ||= 'fuga'
b
```
```
実行結果: fuga
```
```
# 存在しない場合
b = nil
b ||= 'fuga'
b
```
```
実行結果: fuga
```
