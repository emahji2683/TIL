# swiftにおける変数・定数

swiftにおいては、変数はvar(variableの略)、let(略ではなくそのまま)
使用方法は以下に示す。

```
var 変数名: 値
let 定数名: 値
```

# 実際の使用方法
## 概要
変数・定数を定義して、出力してみる。

## 具体のコード

コード
```
let st = "name_is"
var name = "maika"

print(st)
print(name)
```

出力内容
```
└[~/TIL/20250118/swift]> swift variable.swift
name_is
maika
```
